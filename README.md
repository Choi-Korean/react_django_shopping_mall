# react_django_shopping_mall

## 설명
이 프로젝트는 DRF와 React로 동적 Shopping mall 구현을 목표로 하고 있습니다.

## 목표
* Django API를 통한 data 송수신
* React를 통해 API server에 data 송수신
  * Frontend 비동기 데이터 처리
